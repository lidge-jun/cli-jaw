CLI-JAW runs natively inside the Windows Subsystem for Linux (WSL). Follow these two simple steps to get your AI assistant up and running.

1

Install WSL

Open PowerShell as an Administrator (Right-click Start menu > Windows PowerShell (Admin)) and run this command. It will install the required Linux components.

wsl --install
Attention: Your computer will prompt you to restart. Please restart your computer before proceeding to Step 2.
2

Bootstrap CLI-JAW

After restarting your PC, open Ubuntu from the Start Menu. Create a UNIX username and password if prompted, then paste this one-liner script:

curl -fsSL https://raw.githubusercontent.com/lidge-jun/cli-jaw/master/scripts/install-wsl.sh | bash
What this does:
1. Installs Node.js v22 (using fnm or nvm)
2. Installs cli-jaw globally via npm
3. Runs a diagnostic check
3

Authenticate AI Engines

CLI-JAW needs at least one AI engine to work. Pick whichever you have and run its auth command in the terminal:

4

Start Chatting

You're all set! Start the CLI-JAW local server right from your Ubuntu terminal.

jaw serve
